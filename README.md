# Ptr

Ptr: A ModSharp libary for simplify development.

## Installation

Go to [releases](https://github.com/laper32/Ptr/releases) and download the latest release.

> [!NOTE]
> Always download latest release.

## Usage

In your Module's csproj, Add the following PackageReference:
```xml
<PackageReference Include="Ptr.Shared" Version="*" PrivateAssets="all"/>
```
